    템플릿 작업 가이드

    1. 템플릿 작업을 위해 관리자 사이트(사이트도메인/admin)로 접속 (예: https://soundrank.net/admin)

    2. 관리자 사이트 로그인

      * 기본 계정 정보 ID : admin / Password : a000000! (해당 계정으로 로그인 되지 않을 경우 퍼블리셔팀 팀장 또는 작업배치자에게 문의)

    3. 로그인 후 상단메뉴 → 디자인설정으로 이동

    4. 좌측 메뉴 설명

      • PC/모바인디자인설정 : PC버전의 경우 PC디자인설정 / 모바일버전의 경우 모바일디자인설정에서 작업

      • PC/모바일디자인설정 → 스타일관리 : PC버전/모바일버전에서 작업한 템플릿들이 저장된 곳

        * 신규스타일 등록으로 새로운 템플릿을 등록하거나 기존에 등록된 템플릿 확인용으로 사용

      • PC/모바일디자인설정 → 메인페이지관리 : PC버전/모바일버전 메인페이지 작업 시 사용

      • PC/모바일디자인설정 → 서브메인페이지관리 : PC버전/모바일버전 서브페이지를 템플릿으로 작업 시 사용

        * 서브페이지는 특별히 서브메인페이지로 요청받는게 아니라면 FTP서버에서 파일생성으로 작업합니다.

      • PC/모바일디자인설정 → 인트로페이지관리 : PC버전/모바일버전 메인페이지 접속 전 페이지 작업 시 사용

        * 특별히 요청 받는게 아니라면 작업하지 않습니다.

      • PC/모바일디자인설정 → 서브상단부분관리 : 모든 서브페이지 상단부분에 필요한 요소가 있을 때 작업

        * 서브페이지에서도 메인페이지 헤더를 사용할 경우, 햄버거버튼 팝업, 검색버튼 팝업 등이 위치합니다.

      • PC/모바일디자인설정 → 서브하단부분관리 : 모든 서브페이지 하단부분에 필요한 요소가 있을 때 작업

        * 푸터, 하단메뉴 등이 위치합니다.

    5. 신규 템플릿 생성

      1. PC 또는 모바일 스타일관리 → 신규스타일 등록
      2. 스타일명 : 사이트이름 + 직관적으로 작명 (예 : 사운드랭크 메인헤더 / 사운드랭크 왜 음원이 중요할까?! )
      3. 샘플이미지 : 디자인시안에서 작업하는 부분만 1배 크기로 저장해서 등록
      4. 상품포함여부 : 상품정보가 들어가는 템플릿의 경우 포함 체크 후 스타일 상품개수에 노출이 필요한 상품개수 등록 (그 이외에는 등록할 필요없음)
      5. 배너포함여부 : 해당 템플릿에 이미지가 필요할 경우 포함 체크 후 배너 개수에 필요한 이미지 개수 등록
      6. 반복HTML : 상품정보가 들어가는 템플릿의 경우 스타일 상품개수에 등록된 숫자만큼 노출되게 하기 위한 기능
      7. 스타일 템플릿 : 해당하는 템플릿에 CSS, HTML, Javascript 등록 (Visual Studio Code에서 작업 후 복사 붙여넣기 하는 방식으로 작업하면 편함)
